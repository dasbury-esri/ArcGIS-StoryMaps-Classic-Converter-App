# ArcGIS Maps SDK for JavaScript Resources

Where to find help for working with the ESRI [ArcGIS Maps SDK for JavaScript](https://developers.arcgis.com/javascript/latest/). 

> The library is commonly referred to internally as the **JS SDK**, previously **JS API**.

## Documentation

- Overview: https://developers.arcgis.com/javascript/latest/
- Maps, Scenes, Layers: https://developers.arcgis.com/documentation/mapping-apis-and-services/maps/
- Maps: https://developers.arcgis.com/documentation/mapping-apis-and-services/maps/maps-2d/
- Data Layers: https://developers.arcgis.com/documentation/mapping-apis-and-services/maps/data-layers/
- Basemap Layers: https://developers.arcgis.com/documentation/mapping-apis-and-services/maps/basemap-layers/
